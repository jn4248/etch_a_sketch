Etch-a-Sketch Drawing Pad

The drawing canvas can be created using either of two methods:

Method 1: The grid of tiles is created by adding each tile individually over two for-loops.

Method 2: The grid of tiles is created using a for-loop to build a template row, and then cloning the template to build each additional row.

While both methods produce a result with the same functionality, method 2 contructs the canva noticeably faster than Method 1 when there are more than 50 rows and columns.
